## Why 1.4 LTS

EM 1.4 is the version we recommend for **production projects** that need
to remain readable and tooled for years without surprises. It is pinned
to Blender 4.3 LTS, which Blender Foundation supports until mid-2026,
giving the combination a long, stable shelf life.

If you are starting a new excavation, a thesis, a multi-year EU-funded
project, or you need to deposit data on Zenodo with a guarantee of
reproducibility — 1.4 LTS is your friend.

## What's in 1.4

- **TSU (Transformation Stratigraphic Unit)** introduced.
- Refinements across the formal language and consolidation toward 1.5.
- Stable Stratigraphy Manager, Visual Manager, Epochs Manager.
- Heriverse export pipeline (early form).

## Compatibility

| Component | Required |
|---|---|
| Blender | **4.3 LTS** (pinned — don't upgrade Blender mid-project) |
| yEd | 3.21 or later |
| OS | Windows · macOS (Apple Silicon + Intel) · Linux |

## Installing

1. Install **Blender 4.3 LTS** from
   [blender.org/download/lts](https://www.blender.org/download/lts/).
2. Download the EM Tools 1.4 release from
   [GitHub Releases](https://github.com/zalmoxes-laran/EM-blender-tools/releases).
3. In Blender: `Edit → Preferences → Add-ons → Install` and select the `.zip`.

## Support window

LTS support means: bug-fix releases as long as Blender 4.3 LTS itself is
maintained by Blender Foundation. Plan migrations to 1.5 toward the
end of 2026 once 1.5 reaches stable.
