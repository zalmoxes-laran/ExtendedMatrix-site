This is a stub. Delete it once you have at least 2–3 real slides.

**How to add a slide**

1. Drop a high-resolution image (1600px wide minimum) in
   `src/assets/showcase/<your-slug>.jpg`.
2. Create a copy of this file named `<your-slug>.md` and update the
   frontmatter: `title`, `image`, `imageAlt`, optional `caption`, optional
   `project` (linking to a project slug), `featured` (true if it should
   appear in the homepage rotator), and `order`.
3. The body (this part below the frontmatter) is optional and will only
   show on the dedicated `/showcase/<slug>` page — not on the homepage.
